### 문제 설명

<p>한양 나라에는 $N$개의 마을과 $M$개의 도로가 있다. 각 마을에는 $1$부터 $N$까지의 서로 다른 번호가 붙어 있으며, 각 도로는 서로 다른 두 마을을 양방향으로 연결한다. 초기에 모든 마을에는 아무것도 건설되어 있지 않다.</p>

<p>어떤 마을의 <strong>이웃 마을</strong>이란, 그 마을과 도로로 직접 연결된 마을을 의미한다.</p>

<p>당신은 다음 쿼리 $Q$개를 처리해야 한다.</p>

<ul>
<li><code>1 i</code>: $i$번 마을에 지하철 역을 건설한다.</li>
<li><code>2 i</code>: $i$번 마을의 이웃 마을 중 지하철 역이 하나 이상 건설된 마을이 몇 개인지 출력한다.</li>
</ul>

### 입력 

 <p>첫째 줄에 마을의 수 $N$, 도로의 수 $M$, 쿼리의 수 $Q$가 공백으로 구분되어 주어진다. ($2 \le N \le 200\,000$; $1 \le M \leq \min \!\left( \frac{N(N-1)}{2}, 200\,000 \right)$; $1 \le Q \le 200\,000$)</p>

<p>이후 $M$개의 줄에 걸쳐 각 도로가 연결하는 두 마을의 번호 $u, v$가 공백으로 구분되어 주어진다. ($1 \leq u, v \leq N; u \ne v$)</p>

<p>이후 $Q$개의 줄에 걸쳐 각 쿼리를 나타내는 정수 $q, i$가 공백으로 구분되어 주어진다. ($1 \le q \le 2$; $1 \le i \le N$)</p>

<p>두 마을을 연결하는 도로의 수는 최대 $1$개이다. </p>

<p>2번 쿼리는 한 번 이상 주어진다.</p>

### 출력 

 <p>각 2번 쿼리에 대해 쿼리의 결과를 한 줄에 하나씩 출력한다.</p>
